First Example

This layout uses fluid columns. The paragraph text reflows to to fill the number of columns

Wotcha

This layout is built to demonstrate the use of a fluid column structure with a standfirst and article text. The structure is based on This. and built using haml and scss as part of my experiments in responsive design.

When resized the layout will change from 3 columns down to 2, then 1 and the article text will reflow within those columns.
